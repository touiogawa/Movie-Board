<最近映画を見る中で実際に面白いのかどうか調べるために
レビューサイトを見るのだが少し不便だなと思うところもあった為
自分なりに使いやすいサイトを作ったらおもしろそうだなと感じたため>

### 主な利用シーン

 ・面白そうな映画を探している際に>
